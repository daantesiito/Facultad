	modos de direccionamientos, y a tecnicas como SIMD y MIMD paralelismo, etc.

An instruction set architecture (ISA) describes the aspects of a computer architecture
visible to a programmer, including the native datatypes, instructions, registers, addressing
modes, memory architecture, etc.).
An ISA includes a specification of the set of all binary codes that are the native form of
commands implemented by a particular CPU design. The set of these codes for a
particular ISA is also known as the machine language for the ISA.
"Instruction set architecture" is sometimes used to distinguish this set of characteristics
from the microarchitecture, which is the set of processor design techniques used to
implement the instruction set. Computers with different microarchitectures can share a
common instruction set.

Una arquitectura de conjunto de instrucciones (ISA) describe el aspecto en que una arquitectura de computadoras es visible al programador,
incluyendo los tipos de datos nativos, intrucciones, registros, modos de direccion, arquitectura de memoria, etc.
Una ISA incluye unas especificaciones del conjunto de todos los codigos binarios que son la forma nativa de los comandos implementados por el diseño de CPU particular.
El conjunto de estos codigos para una ISA en particular es tambien conocida como el lenguaje de maquina para esa ISA.
"Arquitectura de conjunto de instrucciones" es a veces usado para distinguir el conjunto de caracteristicas de la micro arquitectura, que es el conjunto de tecnicas de diseño de procesador
usado para implementar el conjunto de intrucciones.
Las computadoras con diferentes micro arquitecturas pueden compartir un conjunto de instrucciones comun.
